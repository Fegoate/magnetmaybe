# magnetmaybe

该项目提供了一个用于演示导弹/火箭弹双站 RCS 插值计算的 MATLAB 界面脚本 `matlab/rcs_app.m`。

## 使用方法
1. 在 MATLAB（或兼容的 Octave 版本）中切换到仓库目录。
2. 运行命令 `rcs_app` 打开界面。
3. 在左侧面板设置频率范围、姿态、发射机/接收机/目标的经纬高等参数。
4. 点击“计算并绘制”按钮，界面会展示：
   - 频率插值后的 RCS 曲线。
   - 方向插值后的 RCS 三维表面，以及入射/散射方向标记。

脚本内部同时完成了地心坐标到弹体坐标的转换，方便在地理坐标下描述双站几何关系。
