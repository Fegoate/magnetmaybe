# magnetmaybe

该项目提供了一个用于演示导弹/火箭弹双站 RCS 插值计算的 MATLAB 界面脚本 `matlab/rcs_app.m`，以及等价的 Java Swing 桌面程序。

## MATLAB 使用方法
1. 在 MATLAB（或兼容的 Octave 版本）中切换到仓库目录。
2. 运行命令 `rcs_app` 打开界面。
3. 在左侧面板设置频率范围、姿态、发射机/接收机/目标的经纬高等参数。
4. 点击“计算并绘制”按钮，界面会展示：
   - 频率插值后的 RCS 曲线。
   - 方向插值后的 RCS 三维表面，以及入射/散射方向标记。

脚本内部完成了地心坐标到弹体坐标的转换，方便在地理坐标下描述双站几何关系。

## Java 使用方法
1. 安装 JDK 17+ 并切换到仓库根目录。
2. 编译：`javac -d java/out $(find java/src/main/java -name "*.java")`
3. 运行：`java -cp java/out rcs.RcsApp`
4. 在左侧输入框调整频率范围、姿态以及发射机/接收机/目标经纬高参数，点击“计算并绘图”。
5. 右侧会同时显示频率方向上的插值曲线和散射方向热力图（绿色圈为入射方向，红色圈为散射方向）。

Java 版本同样实现了地心坐标到弹体坐标的转换，并使用合成 RCS 模型进行频率与方向插值，便于在跨平台环境下演示双站 RCS 计算流程。
